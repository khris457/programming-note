Tag a yang adalah tag hyperlink merupakan singkatan dari anchor (jangkar). Analoginya adalah bahwa jangkar terhubung dengan kapal tertentu, seperti link juga terhubung dengan web tertentu.

Link ditulis dalam atribut tag a yang bernama href, kemudian diisi. Hyperlink bisa diisi dengan teks ataupun gambar. Contoh: <a href="http://www.google.com">ini link</a> (Contoh dengan gambar dapat dilihat pada catatan image).
Jika tag a tidak diberi isi, link tidak dapat diklik. Contoh: <a href="http://www.google.com"></a> tidak dapat diklik (karena tidak muncul apa2 pada halaman HTML).

Tag hyperlink dapat digunakan untuk tiga hal, yaitu external link, internal link / relative url, dan page anchor.

External link: Tag a dapat digunakan untuk menghubungkan link yang berada di luar domain (domainnya berbeda), misalnya http://www.google.com, https://mongosilakan.net/#/translate, dll. Seperti pada contoh tsb, kita harus menuliskan protokol (misalnya http) jika menghubungkan external link.

Internal link: Tag a dapat digunakan untuk menghubungkan link yang berada di dalam domain (domainnya sama), misalnya:
1. halaman2.html (jika berada di folder yang sama, kita bisa langsung menuliskan nama filenya),
2. admin/halaman3.html (tag a akan mencari folder admin di folder tempat dia berada, kemudian mencari file halaman3.html),
3. ../halaman4.html (tag a akan naik satu folder di atasnya, kemudian mencari file halaman4.html),
4. ../user/halaman5.html (tag akan naik satu folder di atasnya, mencari folder user, kemudian mencari file halaman5.html), dll.
Poin 1 merupakan contoh dari internal link, sedangkan poin 2-4 merupakan contoh dari relative url.

Page anchor: Kita dapat menguhubungkan suatu bagian dari suatu web ke bagian lain dari web yang sama, ataupun ke suatu bagian dari web yang lain. Contoh:
1. <a href="#bagian1">Bagian 1</a>, jika diklik, laman web akan bergulir ke bagian web yang mengandung konten dengan id "bagian1". Jika di bawah konten tsb masih tersedia cukup ruang, laman akan bergulir sampai konten yang dituju berada di bagian paling atas layar. Namun, jika di bawah konten tsb tidak tersedia cukup ruang, laman akan bergulir sampai ke bagian paling bawah halaman (sampai mentok.)
2. <a href="halaman3.html#bagian3">di sini</a>, jika diklik, laman akan berpindah ke laman lain, dalam hal ini ke laman halaman3.html, kemudian mencari konten yang memiliki id "bagian3" tepat seperti yang dijelaskan pada bagian 1, begitu juga dengan kondisi2nya.

Hyperlink yang diklik secara default akan mengarahkan kita ke web/bagian web pada tab yang sama. Jika kita ingin membuka tab baru ketika kita mengklik hyperlink, kita dapat mengubah atribut "target" pada tag a. Ada empat nilai pada atribut tsb, yaitu _self (tidak membuka tab baru saat diklik; nilai bawaan), _blank (membuka tab baru saat diklik), _parent, dan _top (untuk parent dan top, ada kaitannya dengan frame, namun frame belum dipelajari untuk saat ini).
