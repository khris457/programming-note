Tag HTML ada yang berpasangan ada yang tidak. Biasanya tag yang berpasangan memiliki tag pembuka dan tag penutup.

Untuk memeriksa atribut lengkap tag HTML, kita bisa kunjungi https://www.w3schools.com/html/html_attributes.asp.

Ada beberapa tag yang bisa kita simpan di dalam tag <head>, yaitu:
judul halaman: <title></title>
CSS: <style></style>
javascript: <script></script>
metadata: <meta></meta>

Metadata itu penting untuk mendeskripsikan website kita.
Metadata terdiri dari: charset (untuk encoding), description, keyword, dan author.
Hal-hal tersebut memudahkan search engine untuk mengidentifikasi website kita.

Tag-tag yang bisa ada di dalam body adalah:
teks: h1-h6 (heading; HTML mendukung sampai enam level heading), p (paragraf), ...
pendukung teks: br (untuk menambahkan baris baru), hr (untuk membuat garis horizontal), em, strong, ...
gambar: img
hyperlink: a (anchor)
list (bullets and numbering): ul (unordered list), ol (ordered list), li (tag li ada di dalam ul maupun ol), dl (definition list), dt, dan dd.
table: table, thead, tbody, ...
form: form, input, select, button, ... (untuk penginputan dsb)
script: script (untuk menyimpan javascript)
object: object (biasanya untuk menyimpan objek multimedia)
grouping: div dan span

Format penulisan tag adalah '<namatag atribut="nilai">' (tanpa tanda petik satu). Cth: <body bgcolor"lightskyblue">.

Sebuah tag bisa memiliki nol atau lebih atribut.

Ada atribut yang dimiliki oleh semua tag, atribut2 ini disebut sebagai atribut global.

Atribut global ada delapan:
accesskey -> untuk mengatur pengaksesan konten menggunakan keyboard
shortcut, class, id, dir -> untuk menentukan arah tulisan
direction, lang -> language, style -> untuk menyisipkan inline CSS
tabindeks -> untuk menentukan di mana kursor mendarat di mana setelah kita menekan tab
title -> untuk memberi judul pada sebuah elemen.

<!-- --> adalah tag komentar. Apapun yang dituliskan di dalam tag tersebut akan diabaikan oleh browser. Tag tersebut berguna untuk devoloper/coder untuk memberi penjelasan pada kodingan, sehingga jika kode dibuka oleh orang lain atau sudah lama tidak dibuka, kode diharapkan bisa dimengerti dengan lebih mudah. Cth: <!-- Ini adalah komentar. -->

Tag HTML tidak case-sensitive.
Ini berarti <P> itu sama saja dengan <p>.
Standar HTML tidak mengharuskan penulisan dengan huruf kecil, tapi organisasi W3C (ada di catatan 1) merekomendasikan penggunaan huruf kecil, dan mengharuskan penggunaan huruf kecil pada tipe dokumen yang lebih ketat seperti XHTML.
