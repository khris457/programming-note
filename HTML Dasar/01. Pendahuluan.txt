HTML: Hypertext Markup Language. Sesuai dengan namanya, html adalah markup language, bukan programming language. Ini dikarenakan html digunakan untuk membuat struktur konten. HTML tidak memiliki fungsi, variabel, dll yang umumnya dimiliki oleh bahasa pemrograman.

HTML pertama kali diciptakan oleh Tim Berners-Lee. Berkatnya, kita bisa mengakses web setiap hari. Tim Berners-Lee juga menciptakan banyak hal selain HTML, seperti HTTP, www, web browser, dan web page. Dia sekarang menjadi direktur dari sebuah organisasi atau konsorsium bernama W3C (World Wide Web Consortium; www.w3.org).

Konsorsium -> himpunan beberapa pengusaha yang mengadakan usaha bersama.

W3C adalah sebuah organisasi yang membuat standar-standar untuk web saat ini.

Detail W3C dapat dilihat pada lamannya www.w3.org -> tab about -> FAQ.

Kegiatan utama W3C adalah mengembangkan protokol dan pedoman yang menjamin perkembangan web jangka panjang.

W3C tidak memiliki kantor pusat tunggal. Bagaimanapun, ada empat institusi yang menjadi "tuan rumah" W3C, yaitu MIT (di Cambridge, MA, Amerika Serikat), ERCIM (di Sophia-Antipolis, Perancis), Keio University (dekat Tokyo, Jepang), dan Beihang University (di Beijing, Cina). Jadi orang-orang pintar di seluruh dunia bergabung untuk menciptakan standar-standar di dunia web.

Standar-standar yang diciptakan antara lain desain web dan aplikasi, arsitektur web, web semantik, dll (dapat dilihat pada lamannya di tab standards).

Kita dapat melihat evolusi web di www.evolutionoftheweb.com (walaupun sekarang situs tsb sedang error 500, kesalahan server).

HTML, CSS, dan Javascript punya tugas masing-masing. Jadi HTML itu digunakan untuk membuat struktur halaman web, CSS digunakan untuk menghias halaman web tersebut, sedangkan Javascript yang merupakan bahasa pemrograman digunakan untuk interaktivitas pada web.

Analogi HTML, CSS, dan Javascript adalah sebagai berikut:
HTML adalah maneken (mannequin) yang polos. Ia memiliki kepala, tangan, kaki, dll.
CSS adalah baju, aksesoris, dsb yang digunakan untuk menghias maneken tersebut.
Sedangkan Javascript ibarat chipset (semacam otak robot) yang membuat maneken tersebut bisa bergerak/berinteraksi dengan lingkungan sekitarnya.

Kita bisa melihat perkembangan website seperti amazon dan facebook untuk mengetahui contoh dari perkembangan web.

Ada website yang murni dibuat dengan hanya menggunakan HTML, CSS, dan Javascript, yaitu www.pw1.if-unpas.org/2014. Website ini tidak menggunakan bahasa pemrograman server side seperti php, hsp, dsb atau DBMS seperti MySQL.

Kode HTML bisa langsung ditulis tanpa format <!doctype html> dst.
