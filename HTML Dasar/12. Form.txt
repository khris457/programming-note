Form adalah tempat untuk memasukkan data pengguna. Begitu juga dengan form pada HTML.

Untuk membuat form, kita perlu menggunakan tag <form></form> supaya HTML tahu bahwa kumpulan elemen-elemen yang akan kita tuliskan merupakan bagian dari suatu form.

Lima elemen form yang sangat sering digunakan dan akan dipelajari adalah:
1. input -> untuk menerima input dari user (elemen ini merupakan elemen yang paling penting).
2. textarea -> untuk menerima input teks yang ukurannya panjang. Elemen input juga bisa menerima teks, namun untuk teks yang panjang, lebih baik menggunakan textarea.
3. select -> digunakan untuk membuat drop down button atau combo box.
4. button -> untuk melakukan tugas spesifik ketika ditekan.
5. label -> untuk memberi label/nama pada elemen input.

Tag input tidak memiliki penutup. Tag input memiliki atribut tipe (type) dengan paling sedikit ada tujuh nilai, di antaranya:
1. text -> untuk menerima input teks.
2. password -> untuk menerima input password. Mirip seperti text, namun apapun yang diketikkan pada kolom password akan disensor dengan simbol bulatan hitam.
3. radio -> untuk menerima input semacam pilihan ganda (yang mana kita hanya bisa memilih satu dari sekian pilihan yang ada).
4. checkbox -> untuk menerima input yang memiliki beberapa pilihan seperti radio. Namun bedanya, kita dapat memilih lebih dari satu pilihan.
5. submit* -> untuk mengirimkan data yang telah dimasukkan pengguna.
6. reset* -> untuk menghapus semua data yang telah dimasukkan pengguna.
7. button* -> untuk melakukan tugas spesifik ketika ditekan.
*Nilai submit, reset, dan button tidak disarankan untuk digunakan, karena lebih baik menggunakan tag button yang lebih sesuai dengan peruntukkannya.
Contoh penulisan tag input:
---------------------------
Username: <input type="text" id="username">
---------------------------
Penulisan seperti di atas sebenarnya tidak disarankan karena ada teks (elemen) yang tidak menggunakan tag.
Sebaiknya kita membiasakan penulisan elemen pada HTML menggunakan tag, agar HTML tahu apa sebenarnya elemen yang kita tuliskan.
Penulisan teks yang disarankan adalah menggunakan tag label, yang akan dijelaskan setelah ini.

Kita dapat menuliskan nama pada tag label di dalam tag label tsb.
Tag label umumnya dituliskan menggunakan atribut for, yang akan dihubungkan ke tag lain yang memiliki id.
Nilai atribut for pada tag label harus sama dengan nilai id pada tag yang ingin dihubungkan. Contoh:
---------------------------
<label for="username">Username : </label>
<input type="text" id="username">
---------------------------
<label for="password">Password : </label>
<input type="password" id="password">
---------------------------
<input type="checkbox" id="menyanyi"><label for="menyanyi">menyanyi</label>

Untuk tag radio, kita harus menggunakan atribut name, lalu, nilai pada atribut tersebut harus sama untuk tombol2 radio yang merupakan satu kesatuan. Contoh:
---------------------------
<input type="radio" name="jenis_kelamin" id="pria"><label for="pria">pria</label>
<input type="radio" name="jenis_kelamin" id="wanita"><label for="wanita">wanita</label>

Tag submit, reset, maupun button dapat kita beri nama sesuai keinginan kita. Nama tsb akan ditampilkan pada halaman website. Contoh:
---------------------------
<input type="submit" value="mantap!">

Tag button memiliki penutup. Contoh penulisannya:
---------------------------
<button type="button">Kirim Dong</button>
---------------------------
Seperti pada contoh di atas, kita dapat menuliskan nama tombol di dalam tag button. Nama tsb akan ditampilkan di halaman HTML.

Umumnya, penulisan spasi pada tag HTML tidak akan berpengaruh pada tampilan di halaman website kalau spasinya lebih dari satu (spasi pertama masih berpengaruh).

Kiat: Kita bisa menambahkan spasi dengan beragam lebar. Berikut caranya:
Satu spasi – Tikkan &nbsp;
Dua spasi – Tikkan &ensp;
Empat spasi – Tikkan &emsp;
Penambahan spasi ini bisa digunakan untuk menyejajarkan tanda titik dua (:) dalam penulisan label pada form.
